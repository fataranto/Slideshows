Welcome!
Mathigon Slideshows is a powerful framework for creating HTML slideshows on the web. It was initially created as part of the Mathigon project to make engaging mathematics lessons that can be watched by students or be presented by teachers in classrooms.

Click anywhere on the slide to show the toolbars and use the top buttons to toggle narration and presenter notes. Use the keyboard arrows to navigate; on mobile devices simply swipe.

Features

Slides can consist of any HTML content, including videos, forms or iframes. They can be customised entirely using JavaScript and CSS.

You can embed spoken narration as well as presenter notes, and users and disable these independently.

Mathigon Slideshows are fully mobile and WebApp capable. They fluidly adjust to the window or screen size: try resizing your browser!

With Mathigon Slideshows included are many amazing effects, animations, transitions and designs. We are constantly adding more, and it is easy to customise them in JavaScript.

You can navigate the slideshow using keyboard shortcuts, the toolbars, a slider or mobile gestures.

You can share slideshows via Facebook, Twitter and other Social Sites without leaving the slideshow, or embed them in your website or blog.

You can link to individual slides by using hashes in the URL.

And the slideshows even have a full screen button.

This presentation gives an overview of the underlying syntax and how you can create your own slideshows. Remember to download all source files from mathigon.org/slideshows.

The Basic Syntax

Every slide is represented by a new section HTML tag, as shown below:

<div id="stage" data-ratio="0.75">

  <section>
    <h1>This is Slide 1</h1>
    <p>And here is some text on slide 1</h1>
  </section>

  <section>
    <h1>This is Slide 2</h1>
    <p>And here is some text on slide 2</h1>
  </section>

<div>

The data-ratio attribute for <stage> gives the slide aspect ratio. It is optional; if missing the slide will wrap to the window width and height.

The <section> tag attributes

There are many additional attributes for the section tags:

<section
    data-title="Tite of Slide"             // slide title; default is no title
    data-transition="fade"                 // transition; default is no transition
    data-duration="2000"                   // transition duration; default is 0
    class="s_shadow"                       // useful for adding css styles to many slides
    style="background-image: url(bg.jpg)"  // add styles to individual slides
  >
      <p>Here is some text</p>
</section>

Since the slide is scaled to fit the browser it is important to get the background style rules right. By default, the background scales to cover the slide. If a tiling image is used, include a background-size rule.
