Welcome! This presentation will give you an overview on the Mathigon Slideshows framework which can be used to create beautiful slideshows for websites.

Press space to continue or click anywhere on the slide to show the toolbars for more options.

Introduction

Mathigon Slideshows is an HTML and JavaScript framework that can be used to create beautiful and widely compatible slideshows on websites. These can take up the entire page or be embedded in smaller iframes.

The framework was initially created to create engaging mathematics lessons that can be watched by students and be presented by teachers.

You can download all source files from mathigon.org/slideshows. An interactive coding-less user interface, as well as the possibility to pass in JSON and XML files rather than HTML, are in preperation.

Some Features of Mathigon Slideshows

The slideshows can consist of any HTML content. Styles, animations and effects can be customised entirely using CSS and JavaScript.

Some other features include

  • Many template slide designs, backgrounds, images, transitions and effects, including everything used in this slideshow,
  • Presenter notes and spoken narration which can both be disabled by the user,
  • Full screen, where supported by browser,
  • Mobile and mobile app capable, including a fluid layout adjusting to the browser size,
  • Preloading of images in slides nearby.

The Basic Syntax

Every slide is represented by a new section HTML tag, as shown below:

<div id="stage" data-ratio="0.75">

  <section>
    <h1>This is Slide 1</h1>
    <p>And here is some text on slide 1</h1>
  </section>

  <section>
    <h1>This is Slide 2</h1>
    <p>And here is some text on slide 2</h1>
  </section>

<div>

The data-ratio attribute for <section> is optional. If missing the slide will always wrap to the window width and height.
